Basics of Internet coding

HTML

HTML is the standard markup language for creating webpages

  • HTML describes the structure of a web page, and consists of a series of elements.
  • HTML elements tell the browser how to display content.
    Tutorial on HTML

    CSS

    CSS is a language that describes how HTML elements are to be displayed on screen, paper, or other media.

  • CSS saves a lot of work, because it can control the layout of multiple web pages all at once.

    Javascript

    Javascript is the programming language of HTML and the web

  • Javascript can change
  • Javascript can show and hide the elements of HTML and more
